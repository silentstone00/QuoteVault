---

## Tips for Demo

1. **Practice the flow** - Know exactly which buttons to tap
2. **Have test data ready** - Pre-create some favorites and collections
3. **Keep it moving** - Don't linger too long on any one feature
4. **Show, don't tell** - Let the UI speak for itself
5. **End strong** - Emphasize the AI development aspect
6. **Time yourself** - Aim for 3-4 minutes total
